# GitHub博客部署指南

## 📋 准备工作

你需要的文件已经准备好了：
- `index.html` - 博客主页
- `README.md` - 项目说明
- `部署指南.md` - 这个文件

## 🚀 详细部署步骤

### 第1步：登录GitHub
1. 打开 https://github.com
2. 使用你的邮箱 `3420879706@qq.com` 登录
3. **重要**: 登录后立即修改密码！

### 第2步：创建新仓库
1. 点击右上角的 "+" 号
2. 选择 "New repository"
3. 仓库设置：
   - **Repository name**: `my-personal-blog`
   - **Description**: `我的个人博客`
   - **Public** ✅ (必须选择公开)
   - **Add a README file** ❌ (不勾选，我们有自己的文件)

### 第3步：上传文件
1. 创建仓库后，点击 "uploading an existing file"
2. 将以下文件拖拽上传：
   - `index.html`
   - `README.md`
   - `部署指南.md`
3. 在底部填写提交信息：
   - **Commit message**: `初始化个人博客`
4. 点击 "Commit changes"

### 第4步：启用GitHub Pages
1. 在仓库页面，点击 "Settings" 选项卡
2. 向下滚动找到 "Pages" 部分
3. 在 "Source" 下选择：
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. 点击 "Save"

### 第5步：访问你的博客
等待2-5分钟后，你的博客将在以下地址可用：
```
https://你的用户名.github.io/my-personal-blog
```

例如，如果你的GitHub用户名是 `zhangsan`，那么博客地址就是：
```
https://zhangsan.github.io/my-personal-blog
```

## 📝 如何编辑博客

### 添加新文章
1. 在GitHub仓库中点击 `index.html`
2. 点击编辑按钮（铅笔图标）
3. 找到现有文章，复制这个结构：

```html
<article class="post">
    <h2 class="post-title">你的文章标题</h2>
    <div class="post-meta">发布时间: 2025年10月17日</div>
    <div class="post-content">
        <p>你的文章内容...</p>
        
        <h3>小标题</h3>
        <p>更多内容...</p>
        
        <!-- 添加图片 -->
        <img src="https://图片链接" alt="图片描述">
        
        <!-- 添加表格 -->
        <table>
            <tr><th>列1</th><th>列2</th></tr>
            <tr><td>内容1</td><td>内容2</td></tr>
        </table>
    </div>
</article>
```

4. 修改内容后，滚动到底部
5. 填写提交信息，点击 "Commit changes"
6. 等待几分钟，网站自动更新

### 上传图片
1. 在仓库中创建 `images` 文件夹
2. 上传图片文件
3. 在文章中使用：`<img src="images/图片名.jpg">`

## 🆘 常见问题

**Q: 网站显示404错误？**
A: 等待5-10分钟，GitHub需要时间部署。

**Q: 修改后网站没更新？**
A: 清除浏览器缓存，或等待几分钟。

**Q: 图片不显示？**
A: 检查图片链接是否正确，确保图片已上传到仓库。

## 📞 需要帮助？

如果遇到问题，可以：
1. 检查GitHub Pages设置
2. 查看仓库的Actions页面是否有错误
3. 确认所有文件都已正确上传
