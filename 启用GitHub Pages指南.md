## 📋 现在需要启用GitHub Pages：

### 第1步：进入仓库设置
1. 访问你的仓库：https://github.com/feng1201/my_blog
2. 点击仓库顶部的 **"Settings"** 选项卡

### 第2步：找到Pages设置
1. 在左侧菜单中向下滚动
2. 找到并点击 **"Pages"**

### 第3步：配置Pages
1. 在 **"Source"** 部分：
   - 选择 **"Deploy from a branch"**
2. 在 **"Branch"** 部分：
   - 选择 **"main"**
   - 文件夹选择 **"/ (root)"**
3. 点击 **"Save"** 按钮

### 第4步：等待部署
- GitHub会显示一个绿色的提示框
- 等待2-5分钟让GitHub处理

### 第5步：访问你的博客！
你的博客将在以下地址可用：
```
https://feng1201.github.io/my_blog
```

## 📝 如何编辑博客：

### 添加新文章：
1. 在GitHub仓库中点击 `index.html`
2. 点击编辑按钮（铅笔图标）
3. 复制现有的文章块：
```html
<article class="post">
    <h2 class="post-title">新文章标题</h2>
    <div class="post-meta">发布时间: 2025年10月17日</div>
    <div class="post-content">
        <p>文章内容...</p>
    </div>
</article>
```
4. 修改标题和内容
5. 提交更改

### 上传图片：
1. 点击 `images` 文件夹
2. 点击 "Add file" → "Upload files"
3. 拖拽图片文件
4. 在文章中使用：`<img src="images/图片名.jpg">`
