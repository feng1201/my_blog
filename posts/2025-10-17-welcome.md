# 欢迎来到我的博客！

这是我的第一篇博客文章。这个博客系统有以下特色功能：

## ✨ 主要特性

- **Markdown支持**: 使用Markdown语法写作，简单高效
- **可爱动效**: 页面有飘动的小元素增加趣味性  
- **响应式设计**: 在各种设备上都能完美显示
- **独立文件**: 每篇文章都是独立的Markdown文件
- **标签系统**: 方便文章分类和查找

## 📝 如何添加新文章

1. 在 `posts` 文件夹中创建新的 `.md` 文件
2. 文件名格式：`YYYY-MM-DD-文章标题.md`
3. 在文件开头添加元数据（Front Matter）
4. 使用Markdown语法编写内容

## 🎯 文件示例

```markdown
---
title: "我的新文章"
date: "2025-10-17"
tags: ["技术", "生活"]
author: "你的名字"
---

# 文章标题

这里是文章内容...
```

## 📊 支持的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| Markdown | ✅ | 完整支持 |
| 表格 | ✅ | 如上所示 |
| 代码高亮 | ✅ | 多语言支持 |
| 图片 | ✅ | 本地和网络图片 |
| 数学公式 | 🔄 | 计划支持 |

## 🚀 开始写作

现在你可以开始创建自己的Markdown文件来写博客了！每个文件都是独立的，便于管理和版本控制。

**享受写作的乐趣吧！** ✨
